# 新闻来源网站多选功能使用指南

## 功能说明

工作流支持**多选框**功能，您可以在运行前自由选择需要从哪些新闻来源网站收集新闻。

## 如何使用多选框

### 在工作流界面

1. 打开工作流配置页面
2. 找到"**选择新闻来源网站**"输入项
3. 点击多选框，会显示所有可用的12个新闻来源网站
4. 勾选您需要的网站（支持多选）
5. 点击确认或运行工作流

## JSON配置格式

如果需要在代码中直接指定选择的网站，可以使用以下JSON格式：

### 选择全部网站
```json
{
  "emails": "user@example.com",
  "selected_sites": [
    "toutiao.com",
    "sohu.com",
    "qq.com",
    "163.com",
    "ifeng.com",
    "thepaper.cn",
    "finance.sina.com.cn",
    "sina.com.cn",
    "ylqx.qgyyzs.net",
    "camdi.cn",
    "qxw18.com",

  ]
}
```

### 选择部分网站
```json
{
  "emails": "user@example.com",
  "selected_sites": [
    "toutiao.com",
    "sohu.com",
    "cctv.com"
  ]
}
```

## 推荐组合

### 推荐组合1：全面覆盖（适合日常收集）
- 今日头条、搜狐、腾讯网、网易新闻、澎湃新闻网、央视网
- **特点**：覆盖面广，新闻量充足

### 推荐组合2：权威来源（适合正式报告）
- 澎湃新闻网、央视网、新浪财经、医疗器械行业协会
- **特点**：权威性强，质量高

### 推荐组合3：行业深度（适合专业研究）
- 医疗器械行业协会、东方医疗器械网、新浪财经、腾讯网
- **特点**：行业信息深入

### 推荐组合4：快速浏览（适合日常查看）
- 今日头条、搜狐、网易新闻
- **特点**：新闻量适中，更新快

## 注意事项

1. **至少选择1个网站**：如果未选择任何网站，工作流将报错
2. **新闻数量**：选择的网站越多，可能收集到的新闻越多
3. **新闻质量**：不同网站的新闻质量和更新频率可能不同
4. **去重机制**：无论选择多少个网站，系统会自动去重，避免重复新闻
5. **灵活调整**：可以根据实际需求随时调整选择的网站

## 故障排查

### 问题：多选框无法显示
**解决方案**：
1. 刷新页面
2. 清除浏览器缓存
3. 检查网络连接

### 问题：选择的网站没有收集到新闻
**原因**：
- 该网站暂时没有相关新闻
- 搜索服务暂时不可用
**解决方案**：
- 尝试增加其他网站
- 等待一段时间后重新运行

### 问题：不知道应该选择哪些网站
**建议**：
- 初次使用建议选择全部网站
- 根据实际收到的新闻质量调整
- 参考推荐组合
